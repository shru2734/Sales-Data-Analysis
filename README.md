Sales Data Cleaning and Analysis

This project demonstrates data collection, cleaning, and visualization using Python (Pandas, Matplotlib, Seaborn).

 Contents
- `sales.csv` → Raw dataset
- `Sales_Data_Cleaning.ipynb` → Jupyter/Colab notebook with code
- `clean_sales.csv` → Cleaned dataset

 Tools Used
- Python, Pandas, Matplotlib, Seaborn, Google Colab
